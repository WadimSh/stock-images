# Возврат массива ссылок на файлы хранящиеся в папке

Запрос 

### Request

 + HTTP Method: POST
 
 + URL: https://stock.werstack.com/images/test

 + Body:
```javascript
{
  "image": (binary)
}
```

### Response

 + Status: 200 OK

 + Body:
 ```javascript
{
  "url": "https://stock.werstack.com/images/test/93e4e139-373a-4870-8780-98094844b59b-2.jpg"
}
```
